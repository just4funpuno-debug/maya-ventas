# MAYA Ventas – MVP (Vite + React + Tailwind)

## Requisitos
- Node 18+

## Instalación
```bash
npm install
npm run dev
# build
npm run build
npm run preview
```

## CSVs esperados
**Productos:** `sku, nombre, precio, costo, stock`  
**Ventas:** `fecha, sku, cantidad, precio, vendedora, metodo, cliente, notas`
